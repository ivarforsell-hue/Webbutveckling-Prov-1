📁 Filer & Mappar

Namnge alltid filer/mappar med lowercase, utan mellanslag. Använd bindestreck istället.

  • Startsidan ska heta index.html – webbservern hittar den automatiskt.
  • Stilmall: style.css  |  Skript: script.js
  • Bilder i en mapp t.ex. images/
projekt/
├── index.html
├── style.css
├── script.js
└── images/
    └── bild.jpg

🏗️ HTML-grundstruktur

Varje sida måste ha <!DOCTYPE html>, <html>, <head> och <body>.

<!DOCTYPE html>
<html lang="sv">
 
<head>
  <title>Titel</title>
  <link rel="stylesheet" href="style.css">
  <script src="script.js" defer></script>
</head>
 
<body>
  <!-- Innehåll här -->
</body>
 
</html>
💡 defer = skriptet körs efter att hela HTML:en laddats klart.

🔤 Syntax – Normal & Void element

Normalt element

<p class="intro">Text</p>
  • Starttagg: <p class="intro">
  • Attribut: class = "intro"
  • Innehåll: Text
  • Sluttagg: </p>

Void element (tomt)

<img src="bild.jpg" alt="text">
<input type="text" required>
  • Ingen sluttagg!
  • Booleskt attribut: required – bara närvaron räcker

🎨 CSS-syntax

h1, .rubrik {
  color: #333;
  margin: 10px 20px;
}
  • Selektor – väljer vilket element
  • Deklarationsblock – allt inom { }
  • Deklaration – egenskapsnamn + värde, avslutas med ;
  • Shorthand – t.ex. margin: 10px 20px = topp/botten | vänster/höger

📦 Block vs Inline

Block

Tar hela bredden, börjar på ny rad.

Exempel: div p h1 ul table

Inline

Tar bara sitt eget utrymme, stannar i flödet.

Exempel: span a strong img
